# Barber - A Landing Page Made With Vanilla JS

## Description

This website is a landing page advertising an offline barber training course in
Ukraine.

## Technologies

- Vite (Vanilla JS)
- SaSS
- Email.js
- Notiflix
- Swiper

All images were optimized with Squoosh and are served in WebP format.
